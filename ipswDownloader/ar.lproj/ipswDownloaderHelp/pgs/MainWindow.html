Main Window

* by default you can saw latest Apple TV firmware.

ipswDownloade support multidownloding.

Main controls:

  • First dropdown list is used to select your device model,
  • Second dropdown list is used to select needed firmware version for selected device model,
  • Apple Info button show you official Apple information about selected firmware version,
  • Downloads List button displayed list of current and compleated downloads,
  • Download button will allow you to select where to download selected firmware and download it,

Details panel:

  • Baseband – shows you baseband version coming with the firmware.

  • Jailbreak – shows you the ability of jailbreaking selected device+firmware and a tool to do jailbreak (JB Tools are clickable).
  • Unlock – shows you the ability of unlocking selected device+firmware and a tool to do unlock (JB Tools are clickable).
  • File size – shows you firmware size in megabytes.
  • SHA1 – shows you the SHA1 hash for the firmware to verify its integrity.
  • Build – shows you the official build name of the firmware.
  • Info – shows you useful information about selected firmware+device (if any).
